# Picoshop
A small self-taught  Image processing project.

Picoshop è un progetto personale nato per esplorare i fondamenti dell'elaborazione delle immagini (image processing) attraverso il linguaggio di programmazione C#.

## Descrizione

Il progetto permette di manipolare file grafici tramite un'interfaccia console (CLI). L'utente può interagire con il software per eseguire operazioni di ridimensionamento, applicazione di filtri e altre modifiche strutturali su immagini specificate in fase di input.

## Origine del nome

Il nome Picoshop deriva dalla combinazione di due elementi:
* **Pico**: il prefisso del Sistema Internazionale che esprime il fattore 10^-12 (un milionesimo di un milionesimo).
* **Shop**: un richiamo diretto al celebre software di fotoritocco Adobe Photoshop.

## Caratteristiche principali

* Interfaccia a riga di comando per un controllo diretto e leggero.
* Ridimensionamento delle immagini.
* Applicazione di filtri ed effetti grafici.
* Gestione dei file tramite parametri specificati dall'utente.

## Tecnologie utilizzate

Il progetto è sviluppato in C# e si appoggia a:
* **.NET8.0.
* **SkiaSharp**: una libreria grafica 2D cross-platform basata sul motore grafico Skia di Google, utilizzata per gestire tutte le operazioni di rendering e manipolazione dei pixel.

## Installazione e Utilizzo

1. Clonare la repository locale.
2. Assicurarsi di aver installato il pacchetto NuGet SkiaSharp.
3. Compilare il progetto tramite il comando `dotnet build` o `dotnet Run` .
4. Eseguire l'applicazione e seguire le istruzioni a video per processare le proprie immagini.
